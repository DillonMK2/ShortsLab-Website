How to Export

1. Select your preferred theme and size above

2. Choose a scale factor (2x or higher recommended for crisp exports)

3. Click "Download PNG" to save the logo

For SVG with paths: The SVG files in /public/logo-shortsflow.svg and /public/logo-shortsflow-light.svg now contain path data instead of text, making them font-independent and fully exportable.

For exact Sora font paths: Open this page in a browser, right-click the logo, and use browser dev tools or a tool like Vectr to trace the text. Alternatively, use Figma/Illustrator to type the text in Sora Bold and convert to outlines.
